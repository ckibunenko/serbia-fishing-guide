# Serbia Fishing Guide

Simple static test website in English about recreational fishing in Serbia.

## What this is

This is a disposable niche content site built for testing:

- onboarding flows
- website connection
- SEO/content generation
- publishing flows

It is intentionally lightweight and not connected to a real business.

## How it was built

- Plain static HTML, CSS, and a very small amount of JavaScript
- No framework
- Content-first structure with a homepage, about page, contact page, blog index, and 8 article pages

## File structure

- `index.html` - homepage
- `about.html` - about page
- `contact.html` - contact page
- `blog/` - blog index and article pages
- `assets/styles.css` - site styling
- `assets/script.js` - mobile menu and demo contact form behavior
- `robots.txt` - simple crawler rules

## How to edit content later

1. Open the HTML file you want to edit.
2. Update the text, links, or metadata directly.
3. If needed, adjust styling in `assets/styles.css`.
4. Re-upload the site folder as a ZIP to the same kind of static host or to another simple static hosting platform.
